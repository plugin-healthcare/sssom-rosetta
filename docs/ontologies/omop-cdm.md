# OMOP CDM OWL

- **Name**: OMOP CDM OWL
- **Version**: generated from the OMOP Common Data Model v6 schema
- **Publication**: https://vemonet.github.io/omop-cdm-owl/
- **Source**: https://github.com/vemonet/omop-cdm-owl (includes
  `ontology.ttl`)

OMOP CDM OWL is an OWL ontology generated from the OMOP Common Data Model
(CDM) v6 schema, maintained by OHDSI. It represents the CDM's tables and
fields as classes and properties, providing an RDF-native view of the model
used for observational health data.

Pinned version, checksum, and download details are recorded in
`src/sssom_rosetta/ontology/sources.py` once that registry exists.
