# ONZ-G

- **Name**: Een Ontologie voor de Nederlandse Zorg — Generiek (ONZ-G)
- **Version**: 2.8.1
- **IRI**: `http://purl.org/ozo/onz-g`
- **Publication**: https://kik-v-publicatieplatform.nl/ontologie/onz-g/2.8.1
  (TTL, OWL, and JSON-LD downloads are linked from that page: `ontology.ttl`,
  `.owl`, `.jsonld`)

ONZ-G is a generic ontology for Dutch healthcare, developed as part of the
KIK-V program. It defines the core classes and properties used to describe
care organizations, clients, and related concepts in a shared, formal
vocabulary.

Pinned version, checksum, and download details are recorded in
`src/sssom_rosetta/ontology/sources.py` once that registry exists.
